Instructions:

    1. Update Notepad++ (tested on v6.6.7).

    2.  a.  For installer version of Notepad++:
            i.  Place functionList.xml and userDefineLang.xml in %APPDATA%\Notepad++\
            ii. Place plugins\APIs\Nasal.xml in <install_folder>\plugins\APIs with the other language XML files
                Example target folder for Nasal.xml: "C:\Program Files (x86)\Notepad++\plugins\APIs"

        b.  For .zip version:   Extract all files into the Notepad++ working directory while maintaining folder hierarchy.
